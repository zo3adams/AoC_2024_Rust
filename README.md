# AoC_2024_Rust
Solutions to the 2024 and 2025 Advent of Code puzzles in Rust

To run a given problem:

```
cargo run --release -- 2025_day01
```


sample_input.txt and input.txt must be changed to the data provided for each problem


build.rs generates aoc_index.rs which contains the list of build targets, last parameter in command above must match entry in that list.
